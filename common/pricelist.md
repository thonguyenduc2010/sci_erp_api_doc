
*** 1.5 Danh bảng giá ***
----
Trả về danh sách bảng giá thuộc thương hiệu (brand) mà token key gọi API
* **URL**

    `GET` Danh sách bảng giá: 
    `/{domain}/api/v1/price-list`
  
    `GET` Danh sách bảng giá theo ID:
    `/{domain}/api/v1/price-list/<id>`
  

* **Method:**

    `GET`

* **Url Params**
  
  | Attribute| Type | Description |
  |---|---|---|
  | id | int  | ID bảng giá |

* **Phản hồi thành công:**
    * **Code:** 200 <br />
    * **Content:** <br />
