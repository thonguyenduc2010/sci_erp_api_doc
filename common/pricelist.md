
***1.4 Danh bảng giá***
----

* **URL**

    `GET` Danh sách bảng giá: 
    `/{domain}/api/v1/price-list`
  
    `GET` Chi tiết danh sách bảng giá theo ID: 
    `/{domain}/api/v1/price-list/<id>`
  
  
* **Method:**

  
    `GET`

* **Url Params**


  | Attribute| Type | Description |
  |---|---|---|
  | id | int  | ID bảng giá |
