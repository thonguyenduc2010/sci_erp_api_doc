
***1.4 Chương trình khuyến mại***
Lấy danh sách chương trình khuyến mại theo thương hiệu và chi nhánh.
----

* **URL**
    /{domain}/api/v1/promotion
  
* **Method:**

    `GET`

* **Url Params**
  | Attribute| Type | Description |
  |---|---|---|
  | company_id | int  | ID chi nhánh |
